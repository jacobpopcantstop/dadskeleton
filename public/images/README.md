# Image Organization

Drop images into the appropriate folders based on which page they'll appear on.

## Folder Structure

### `/home`
- `hero.jpg` - Main hero banner (21:9 aspect ratio recommended)
- `photo-1.jpg` through `photo-6.jpg` - Photo grid images (4:3 aspect ratio)
- `team.jpg` - Bottom team photo banner (3:1 aspect ratio)

### `/about`
- `troupe.jpg` - Hero troupe photo (21:9 aspect ratio)
- `sidebar-1.jpg` - Sidebar photo (3:4 aspect ratio)
- `sidebar-2.jpg` - Sidebar photo (square)
- `headshot-1.jpg` through `headshot-8.jpg` - Team member headshots (square)
- `performance.jpg` - Bottom performance photo (3:1 aspect ratio)

### `/calendar`
- `hero.jpg` - Live show photo banner (21:9 aspect ratio)
- `show.jpg` - Show photo (4:3 aspect ratio)
- `venue.jpg` - Venue photo (square)
- `audience.jpg` - Audience photo (4:3 aspect ratio)
- `past-1.jpg` through `past-4.jpg` - Past show photos (4:3 aspect ratio)

### `/gallery`
All images should be square. Name them descriptively:
- `skeleton-wars-01.jpg`
- `improv-night-02.jpg`
- `behind-scenes-03.jpg`
- etc.

### `/get-involved`
- `community.jpg` - Community photo banner (3:1 aspect ratio)
- `sidebar.jpg` - Sidebar photo (square)
- `grid-1.jpg` through `grid-4.jpg` - Bottom grid photos (square)

### `/videos`
- `thumbnail-1.jpg` through `thumbnail-4.jpg` - Video thumbnails (16:9 aspect ratio, optional)

### `/bone-zone`
- `wallpaper-desktop.jpg` - Desktop wallpaper
- `wallpaper-mobile.jpg` - Mobile wallpaper
- Any other secret content images

## Image Guidelines

- Use JPG for photos, PNG for graphics with transparency
- Optimize images for web (compress before uploading)
- Recommended max width: 1920px for heroes, 800px for thumbnails
- Keep file sizes under 500KB when possible
